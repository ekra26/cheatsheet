HTML REFERENCES

Table Tags

These tags are used to represent tabular data (rows and columns containing data) in HTML.

The table below was created using these tags! Feel free to press inspect to see them in action.

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head Defines column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table data container.
<tfoot> Table Foot The table footer container.

CSS REFERENCES

CSS Selectors

Selectors are ways in which you can select elements from HTML to be styled using CSS.

Selector Name HTML CSS Description
Type Selector <h1> h1 {} Selects elements according to the HTML tag.
ID Selector <...id="nameid"/> #nameid {} Selects elements based on the value of its id attribute.
Class Selector <...class="classname"/> .a {} Selects elements based on the value of their class attribute.
